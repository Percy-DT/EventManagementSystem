# Event Management System (Java)

## Overview
This is a desktop-based Event Management System developed using Java Swing. The application allows users to manage events by adding, viewing, searching, and storing event details efficiently.

##  Features
- Add new events (title, category, date)
- View all saved events
- Search events by category
- Store event data in a text file
- Simple and user-friendly graphical interface

## Technologies Used
- Java
- Java Swing (GUI)
- File Handling (BufferedReader, BufferedWriter)
- Object-Oriented Programming (OOP)

## Project Structure
- `Event.java` – Handles event data and formatting  
- `FileHandler.java` – Manages file storage and retrieval  
- `EventManagerGUI.java` – Main graphical user interface  

## How to Run
1. Open the project in NetBeans, IntelliJ, or Eclipse  
2. Compile all `.java` files  
3. Run `EventManagerGUI.java`  
4. Start adding and managing events
